Spielübersicht
In Monopoly geht es darum, Eigentum zu erwerben und es möglichst gewinnbringed zu vermieten oder zu verkaufen, um so der reichste Spieler zu werden. Die Spieler starten auf dem "Los"-Feld und wandern durch Würfeln im Uhrzeigersinn um das Spielfeld. Landet ein Spieler auf einem kaufbaren Feld, kann er dieses erwerben.
Spieler die Grundstücke besitzen, erhalten Miete von allen anderen Mitspielern die auf das entsprechende Feld kommen. Der Bau von Häusern und Hotels erhöht die Miete beträchtlich, also sollte man möglichst schnell möglichst viele Immobilien bauen. 
Sollte man einmal in Geldmangel geraten, kann man die Grundstücke beleihen und dafür Geld von der Bank erhalten.
Anweisungen von Ereignis- oder Gemeinschaftskarten müssen stets befolgt werden. 
Unter Umständen können Spieler auch im Gefängnis setzen, wo sie einige Runden aussetzen müssen.

Ziel des Spiels
Der reichste Spieler zu werden. Alle anderen Mitspieler müssen dazu in den Bankrott getrieben werden.

Spielablauf
Der Spieler der an der Reihe ist muss würfeln. Die Spielfigur wird um die Summe der gewürftelten Augen bewegt. Es können mehrere Figuren auf einem Spielfeld stehen.
Je nachdem auf welchem Feld ein Spieler landet stehen verschiedene Aktionen zur Auswahl:
Grundstück kaufen
Miete bezahlen
eine Ereignnis- oder Gemeinschaftskarte ziehen
ins Gefängnis gehen
Gehalt von 4000 DM einziehen ("Los"-Feld)

Pasch
Würfelt ein Spieler einen Pasch  (beide Würfeln zeigen die gleiche Augenzahl), muss er erneut würfeln. Ist der erneute Wurf erneut ein Pasch, wird ein drittes Mal gewürfelt. Sollte es sich wieder um einen Pasch handeln,  muss der Spieler ins Gefängnis ohne seinen Zug auszuführen.

Über "Los" gehen
Jedesmal wenn ein Spieler mit seiner Figur das Feld "Los" erreicht oder passiert, wird ihm ein Gehalt von 4000 DM gezahlt.
Sollte eine Figur durch zb. Ein Ereignisfeld zweimal das "Los"-Feld passieren, zieht er zweimal das Gehalt ein.

Landen auf einem Grundstück mit Besitzer
Erreicht ein Spieler mit seiner Spielfigur ein Grundstück welches von einem anderen Spieler besessen wird, bezahlt er automatisch die aktuelle Miete. Diese variert je nach Häuseranzahl. Sollte ein Spieler alle Straßen einer Farbe besitzen verdoppelt sich die Miete. Dies gilt auch für Felder die mit einer Hypothek belastet sind.

Ereignis- oder Gemeinschaftsfelder
Landet man auf einem dieser Felder, wird die oberste Karte des entsprechenden Stapels gezogen. Diese Karten können unter anderem verlangen, dass:
Die Spielfigur bewegt wird
Geld bezahlt werden muss
Geld erhalten wird
man ins Gefängnis muss
man aus dem Gefängnis freikommt.
Eine Gefängnisfreikarte kann für späteren Gebrauch aufgehoben werden.
Sollte die Spielfigur über das "Los"-Feld kommen, erhält man normal sein Gehalt, es sei denn man wandert ins Gefängnis.

Steuerfelder
landet man auf einem dieser Felder, wird der abgezogenen Betrag unter das "Frei Parken"-Feld geschoben. 

Frei Parken
Auf diesem Feld passiert nichts. Landet ein Spieler auf diesem Feld, kassiert er allerdings alles durch Steuern angehäufte Geld.

Häuser bauen
Ein Spieler kann nur dann Häuser auf Straßen bauen, wenn er alle Straßen dieser Farbgrupe besitzt und auf dem Feld steht. Häuser erhöhen die Miete erheblich; der Preis für die Häuser und die entsprechende Miete stehen auf den Besitzkarten der Straßen. Straßen müssen nicht gleichmäßig bebaut werden, eine Straße kann ein Hotel besitzen ohne dass die anderen Straßen der Farbgruppe bebaut wurden.

Hypothek
Wer kein Geld mehr besitzt, aber zb. Miete bezahlen muss, kann sich durch die Aufnahme von Hypotheken Geld beschaffen. 

Bankrott
Sollte ein Spieler in den Ruin getrieben werden, erhält der dafür verantwortliche Spieler alle Besitztümer des Spielers. Der bankrott getriebene Spieler scheidet aus dem Spiel aus und hat verloren.

Ende des Spiels
Das Spiel endet wenn alle bis auf einen Spieler bankrott gegangen sind.
